## 文件操作
  - 引入fs模块
    - 引入fs模块的代码：`var fs=require("fs");`
    - fs模块是nodejs的核心模块，不需要安装，直接引入即可

    console.log(fs);  // 输出fs模块对象
    fs.readFile("./files/1.txt","utf-8",function(err,dataStr){
        if(err){
            return console.log("读取文件失败"+err.message);
        }
        console.log(dataStr);
    })
    第一个参数：文件路径
    第二个参数：读取文件的编码格式
    第三个参数：回调函数

## 模块化编程的概念
- 模块化编程的概念：把一个大程序，拆分成多个小模块，每个模块只完成自己功能的代码，然后再把小模块组合在一起，完成大程序的功能
  一个js文件就是一个模块 那多个模块之间如何通信？
  es6模块规范:
    1. 通过exports导出数据
    2. 通过require引入模块
    例如：
    ```
    1.js
    var name="张三";
     exports{name};
    ```

    ```
    2.js
    import{name} from "./1.js";
    console.log(name);
    ```
    * 注意：
      1. 导出的变量名，必须和引入的变量名一致,导出使用了变量别名，引入的时候必须使用变量别名（exports{name as name2}）
```
  commonjs模块规范:
   // 导出模块
    module.exports = {
    functionName: myFunction
    };

  // 引入模块
    const moduleName = require('./moduleName');
  ```
